Spike Prime Control to Google Colab

Browser Setup (once):

  1. Open this webpage in a Chrome Browser
  2. Go to the URL: chrome://flags
  3. Search "Experimental Web Platform"
  4. ENABLE Experimental Web Platform (#enable-experimental-web-platform-features)
  5. Restart your Chrome web browser
  6. Plug in your SPIKE Prime hub by USB to this computer (and turn it on)!

Once enabled:

Click "Connect" green button and select the port where Spike Prime is connected (e.g., LEGO Technic Large Hub in FS mode...)i>

Then you can:

  • Read data from a sensor
  1. Fill in the public IP adress of the server running in your Google Colab file (e.g., htttp://XXXXX.ngrok.io)
  2. Select the sensor's port you want read data
  3. Select the # of samples and the frequency to acquire them
  4. Click on "Go" button
  • Ask for a file from Colab and run on the Spike Prime:
  1. Fill in the public IP adress of the server running in your Google Colab file (e.g., htttp://XXXXX.ngrok.io)
  2. Write the name of the file without extension
  3. Click on "Ask for file" button
